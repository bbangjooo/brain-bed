# Brain Bed — Growth Channels

## 1. Product-Led Growth (PLG)

### Free vs Premium boundary
| Free (current) | Premium (future) |
|----------------|-----------------|
| BFI tracking | BFI history & trends over time |
| Meditation mode | Custom meditation themes/sounds |
| 3 CLI tools | Plugin system for more tools |
| Basic notifications | Team dashboard (for companies) |
| Local data only | Cloud sync across devices |

### Viral Loop
1. **GitHub stars** — Open source repo, visible on profile
2. **"Powered by Brain Bed"** — Optional watermark on shared BFI screenshots
3. **Social sharing** — "My BFI today: 72 (Heating). Time to rest." shareable card
4. **Word of mouth** — Developer sees colleague's menu bar → "What's that app?"

### Wow Moment
Target: User sees their BFI score for the first time within **2 minutes** of install.
- Install DMG → Drag to Applications → Launch
- App immediately bootstraps from existing CLI logs
- BFI score appears (not starting from 0)
- "Oh, I'm already at Warming?"

## 2. Referral Program (Future)
