# Brain Bed — Market Analysis

## Target Market

**Global** — English-first product targeting developers worldwide who use AI CLI tools daily.

## User Personas

### 1. "Alex" — Solo AI-powered Developer
- **Role**: Freelance full-stack developer
- **Tools**: Claude Code (primary), Cursor, Codex CLI
- **Behavior**: 6-10 hours/day AI-assisted coding, often late nights
- **Pain**: Burns out by Thursday, weekend recovery isn't enough. Knows they should stop but "one more prompt"
- **Channels**: Twitter/X, Hacker News, Reddit (r/programming, r/ExperiencedDevs), Dev.to, GitHub

### 2. "Sam" — Startup CTO / Technical Co-founder
- **Role**: 1-5 person startup, wears many hats
- **Tools**: Claude Code + Gemini CLI + Codex CLI simultaneously
- **Behavior**: Multi-session power user, 3-5 AI tools open at once
- **Pain**: High cognitive load from context-switching between AI tools. Productivity is high but unsustainable
- **Channels**: IndieHackers, Twitter/X, YC community, HN, Discord (startup servers)

### 3. "Jordan" — AI-curious Developer at a Company
- **Role**: Mid-senior developer at mid-size tech company
- **Tools**: Copilot + Claude Code, experimenting with Codex
- **Behavior**: Uses AI tools 3-5 hours/day, increasing rapidly
- **Pain**: Company encourages AI adoption but no guidance on sustainable usage
- **Channels**: Reddit (r/ClaudeAI, r/ChatGPTPro), Slack communities, LinkedIn, company Slack

## Competitive Landscape

| Solution | Approach | Brain Bed Difference |
|----------|----------|---------------------|
| Screen Time (macOS) | Generic app time limits | Not AI-aware, no meditation |
| Freedom / Cold Turkey | Website/app blocking | Can't block CLI tools, no rest experience |
| Pomodoro apps | Fixed intervals | Not intensity-based, mechanical |
| Calm / Headspace | Separate meditation app | No forced entry, requires switching context |
| Stretchly | Break reminders | No AI tracking, no BFI, just timers |

**Brain Bed's moat**: Only product that reads AI CLI logs, calculates cognitive load from actual usage patterns, and forces rest inside the computer.

## Market Size Estimate

- AI developer tool users (Claude Code, Codex, Gemini CLI): ~2-5M globally (2026)
- Heavy users (daily, 4+ hours): ~500K-1M
- Willing to install a desktop utility: ~100K-200K
- **Initial target (first 12 weeks)**: 5,000 installs, 1,000 weekly active
